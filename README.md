# Retail API Parser (Go)

Тестовое задание на разработку парсера розничных сетей на языке Go.

## Особенности реализации
- **API-based:** Сбор данных осуществляется через внутренние API эндпоинты, что обеспечивает стабильность при изменении HTML-верстки.
- **Proxy Support:** Реализована поддержка прокси-серверов через `http.Transport`.
- **Error Handling:** Код обрабатывает блокировки антифрод-систем (WAF/Qrator) и ошибки парсинга JSON.

## Запуск
```bash
go mod init komtek_parser
go run main.go
